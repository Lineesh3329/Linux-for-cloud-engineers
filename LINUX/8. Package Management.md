## Package Management

Package management in Linux is used to install, update, and remove software.

Different Linux distributions use different package managers like apt, yum, or dnf.

It is an essential for managing servers and cloud environments.

### Common Package Managers

- apt     → Used in Ubuntu / Debian
- yum     → Used in older RHEL / CentOS
- dnf     → Used in newer RHEL / Fedora
- rpm     → Low-level package tool

### Basic apt Commands (Ubuntu)
    sudo apt update            - Update package list
    sudo apt upgrade           - Upgrade installed packages
    sudo apt install nginx     - Install package
    
    Example:-
    1. Sudo apt-get install openssh-server - install openssh
    2. Sudo apt-get install vim            - install vim package
    
    sudo apt remove nginx      - Remove package
    sudo apt search nginx      - Search package  

### Basic yum / dnf Commands (RHEL)
    sudo yum install nginx
    sudo yum remove nginx
    sudo yum update

    sudo dnf install nginx
    sudo dnf remove nginx

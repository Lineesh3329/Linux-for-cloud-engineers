Text Editors (vi / vim / nano)
------------------------------
Text editors in Linux are used to create and modify files directly from the command line.

They are mainly used to edit configuration files, scripts, and application settings in servers.

Common editors are vi, vim, and nano.

#### Commands
    vi file.txt        - Open file
    i                  - Enter insert mode
    Esc                - Exit insert mode
    :w                 - Save file
    :q                 - Quit
    :wq                - Save and quit
    :q!                - Quit without saving
    ctrl+shift+c       - copy
    ctrl+shift+v       - paste
Text editors are essential tools for managing Linux servers.

Understanding basic vi or nano commands is important for working in cloud and production environments.

## Task Automation
## Cron (Job Scheduling)
Cron is commonly used for backups, log cleanup, running scripts, and maintenance tasks in servers.
Cron is a Linux utility used to schedule tasks automatically at specific times or intervals.

#### Time Shedule

    Every Min                    
    Every 5 mins

    Every Month

#### Cron Time Format
    
    * * * * * command
    | | | | |
    | | | | └── Day of Week  (0-7)
    | | | └──── Month        (1-12)
    | | └────── Day of Month (1-31)
    | └──────── Hour         (0-23)
    └────────── Minute       (0-59)

#### Main Symbols 
    '*'  - Every
    '/'  - Step
    ','  - Multiple values
    '-'  - Range
    
#### Cron Time cheat sheet

    *              Every
    */2            Every 2 mins
    0              Exact Minute
    0 0 * * *      Everyday at Midnight
    */10 * * *     Every 10 mins
    0*/6 * * *     Every 6 hrs
    0 0 * * *      Everyday at Midnight
    0 0 * * 0      Every Sunday
    0 0 1 * *      Every month on day 1
